# Intel CPU Optimization for GenAI - Key Findings

## Source: Deloitte & Intel Whitepaper - "Advancing GenAI with CPU Optimization"

### Key Insights

**Small Language Models (SLMs) on CPU:**
- Organizations are exploring SLMs that train on smaller, highly curated datasets to solve repeatable, specific problems
- SLMs can run efficiently on CPU architecture without requiring GPUs
- Cost savings of approximately 55% compared to GPU-based solutions
- Deloitte offers cost-efficient solutions helping organizations boost widespread adoption

**Bridging the Gap:**
- As AI grows more complex, demand is rising for SLMs that serve small teams of users and run efficiently on edge devices and compact hardware
- SLMs ideal for on-device AI where space and resources are limited but data security remains a top priority
- Organizations wanting to keep data on internal network or Virtual Private Clouds (VPCs) can run AI on central processing units (CPUs)

**CPU Optimization Benefits:**
- Reduce hardware demands and computational requirements
- Maintain security while delivering cost savings
- Leverage existing CPU-based infrastructure
- Minimize additional expenditures and scale GenAI rapidly
- No need for GPU procurement, implementation, maintenance, and training

### Implications for Enterprise Desktop Deployment
- Modern Intel/AMD CPUs can effectively run SLMs for enterprise use cases
- On-premise deployment maintains data security and compliance
- Significant cost advantages over GPU-based solutions
- Ability to leverage existing desktop infrastructure
